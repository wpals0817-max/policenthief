# 💰 경도 프리미엄 전략 - 은근한 유도

**날짜**: 2026-02-15  
**목표**: 사용자가 자연스럽게 프리미엄의 가치를 느끼고 구매하도록 유도

---

## 🎯 핵심 전략

### 1. 가격 심리
- **₩4,900**: 부담 없는 가격 (커피 2잔 값)
- **일회성 결제**: 구독 부담 없음 → 결정 쉬움
- **방장 결제 = 전체 혜택**: 친구들과 나눠내기 가능
  - 10명이면 1인당 ₩490
  - 20명이면 1인당 ₩245

### 2. 타이밍 전략
**결제 의사가 높아지는 순간을 포착**:
- 방 설정 중 제한에 걸렸을 때
- 대기실에서 인원이 12명 넘을 때
- 게임이 재미있었던 직후 (결과 화면)
- 광고가 방해될 때

## 📍 페이지별 유도 전략

### 메인 화면
**접근**: 부드럽게, 방해 없이

- 광고 배너 (하단, 작게)
- 프리미엄 사용자는 배너 없음 → **차이 체감**
- 프리미엄 배지 (✨PRO) 자랑 → 부러움 유도

**문구 없음** - 시각적으로만 차이 보여주기

---

### 방 설정 화면 ⭐ 핵심
**접근**: 자연스럽게 제한 알리기, 해결책 제시

#### 시나리오 1: 시간 설정
**무료 사용자가 슬라이더를 10분 미만 / 20분 초과로 이동 시**:
```
┌─────────────────────────────┐
│ 게임 시간                    │
│ [=====●=====] 30분 🔒       │
│                              │
│ 💡 프리미엄이면 48시간까지!  │
│    [자세히 보기]             │
└─────────────────────────────┘
```

**문구 톤**:
- ❌ "프리미엄을 구매하세요!"
- ✅ "프리미엄이면 48시간까지 가능해요!"

#### 시나리오 2: 라운드 선택
**무료 사용자가 5 또는 7 라운드 버튼 클릭 시**:
```
라운드: [1] [2] [3] [5🔒] [7🔒]

💡 3라운드 이상은 프리미엄 기능이에요
   [프리미엄 알아보기]
```

**작은 버튼**, 강요하지 않기

#### 시나리오 3: 설정 완료 후
**방 만들기 버튼 근처에 작은 배너**:
```
┌─────────────────────────────┐
│ ✨ 프리미엄 팁               │
│ 방장만 결제해도 방 전체 혜택! │
│ 친구들과 나눠내면 1인당 몇백원 │
│                    [알아보기→] │
└─────────────────────────────┘
```

**위치**: 방 만들기 버튼 위, 작은 정보 카드

### 대기실
**접근**: 상황에 따라 타게팅

#### 시나리오 1: 인원 증가
**인원 10명 도달 시** (아직 12명 안 됨):
```
참여 인원: 10 / 12명

ℹ️ 12명 넘으면 프리미엄이 필요해요
```

**톤**: 정보 제공, 중립적

**인원 12명 도달 시**:
```
참여 인원: 12 / 12명 (정원)

✨ 프리미엄이면 100명까지 가능해요!
   [자세히 보기]
```

**톤**: 긍정적, 가능성 제시

#### 시나리오 2: 방장 설명
**방장에게만 보이는 작은 툴팁**:
```
💡 꿀팁: 방장이 프리미엄이면
         이 방의 모든 사람이 혜택을 받아요!
```

**위치**: 설정 변경 버튼 옆, 작은 물음표 아이콘 클릭 시 표시

### 게임 화면
**접근**: 게임 방해 금지, 최소한만

#### 광고 배너 (무료만)
- 하단 고정, 320x50px
- 게임 중엔 최대한 작게
- **게임 종료 5분 전**: 배너에 작은 텍스트 추가
  ```
  [쿠팡 파트너스 광고]
  💡 프리미엄은 광고 없어요
  ```

#### 근접 알람 발동 시 (선택)
**근접 알람이 여러 번 울린 후**:
```
⚠️ 상대팀이 30m 거리에 있습니다!

(알람 10회 이상 시)
💡 프리미엄은 알람 간격을 더 세밀하게 설정할 수 있어요
```

**선택 사항** - 너무 방해되면 생략

## 📊 전환 유도 시나리오 우선순위

- 게임 재미있었을 때 = 가장 높은 전환율
- 감정적으로 만족한 상태
- 더 하고 싶은 욕구 생김

- 제한에 직접 부딪힘
- 문제 인식 → 해결책 제시

### 3순위: 대기실
- 인원 12명 도달 = 구체적 필요 발생
- 실시간 상황 → 긴급성

### 4순위: 히스토리 화면
- 장기 사용자 = 애착 형성
- 낮은 압박, 자연스러운 제안

### 5순위: 메인 화면
- 시각적 차이만 (광고 유무)
- 직접 유도 없음
