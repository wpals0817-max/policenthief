## 📋 게임 기본 설명 (업데이트)

경찰과 도둑(경도)은 두 팀으로 나눠 경찰이 도둑을 잡아 감옥에 가두고, 도둑은 잡힌 동료를 구출하며 도망치는 술래잡기 놀이입니다.

**승리 조건**:
- **경찰**: 정해진 시간 내 모든 도둑을 검거
- **도둑**: 시간 종료까지 한 명이라도 살아남거나 탈출

**추가 필요**:

- 🆕 **전체 공유**: 모든 플레이어 상태 실시간 표시 (현재 방식)
- 🆕 **경찰만 확인**: 경찰팀만 전체 상황 확인 가능
- 🆕 **비공개**: 아무도 확인 불가 (게임 종료 시에만 공개)
- 🆕 **위치 업데이트 간격**: 몇 분마다 위치 갱신 표시 (실시간/1분/3분/5분/10분)
- 🆕 **근접 알람**: 상대팀이 N미터 이내 접근 시 알림 (10m/20m/30m/50m/100m/OFF)

**구현 위치**: 방 설정 (Room Settings)

**데이터 구조**:
```typescript
interface RoomSettings {
  // 기존...
  captureVisibility: 'all' | 'police-only' | 'self-only' | 'hidden';
  locationUpdateInterval: 'realtime' | 1 | 3 | 5 | 10; // 분 단위
  proximityAlert: 10 | 20 | 30 | 50 | 100 | 'off'; // 미터 단위
}
```

**구현 로직**:
```typescript
// 위치 업데이트 간격 처리
if (settings.locationUpdateInterval === 'realtime') {
  // 기존처럼 실시간 업데이트
} else {
  // N분마다 한 번씩만 위치 표시
  const intervalMs = settings.locationUpdateInterval * 60 * 1000;
  setInterval(() => updateVisibleLocations(), intervalMs);
}

// 근접 알람
function checkProximity(myLocation, otherPlayers) {
  if (settings.proximityAlert === 'off') return;
  
  otherPlayers
    .filter(p => p.team !== myTeam)
    .forEach(enemy => {
      const distance = calculateDistance(myLocation, enemy.location);
      if (distance <= settings.proximityAlert) {
        showProximityAlert(enemy, distance);
      }
    });
}
```
