# 🚀 배포 전 체크리스트

## ✅ 필수 확인 사항

### 1. 게임 기능 테스트
- [ ] **방 생성** - 정상 작동하는지
- [ ] **코드로 입장** - 다른 기기에서 입장 가능한지
- [ ] **GPS 권한** - 위치 권한 요청이 제대로 뜨는지
- [ ] **실시간 동기화** - 위치가 실시간으로 업데이트되는지
- [ ] **팀 배정** - 경찰/도둑이 제대로 배정되는지
- [ ] **체포 시스템** - 거리 기반 자동 체포가 작동하는지
- [ ] **구출 시스템** - 터치 구출이 작동하는지
- [ ] **게임 종료** - 승패 판정과 결과 화면이 제대로 나오는지

### 2. 모바일 환경 테스트
- [ ] **iOS Safari** - 정상 작동하는지
- [ ] **Android Chrome** - 정상 작동하는지
- [ ] **PWA 설치** - 홈 화면 추가 시 아이콘과 스플래시 화면
- [ ] **오프라인** - 오프라인 페이지 표시

### 3. UI/UX 확인
- [ ] **모든 페이지** 라이트 테마 적용 확인
- [ ] **쿠팡 광고** 실제로 보이는지 (여러 페이지에서)
- [ ] **반응형** - 다양한 화면 크기에서 레이아웃 깨지지 않는지
- [ ] **로딩 상태** - 로딩 중 표시가 적절한지

### 4. 성능 확인
- [ ] **페이지 로딩 속도** - 3초 이내
- [ ] **지도 렌더링** - 부드럽게 작동하는지
- [ ] **배터리 소모** - GPS 사용 시 배터리 과다 소모 없는지

### 5. 오류 처리
- [ ] **위치 권한 거부** 시 안내 메시지
- [ ] **네트워크 오류** 시 재시도 버튼
- [ ] **방 코드 오류** 시 명확한 메시지
- [ ] **만료된 방** 접속 시 처리

## 🔍 추가 권장 사항

### 1. 소규모 베타 테스트 (권장)
- 친구 2-3명과 실제 게임 진행
- 야외에서 테스트 (GPS 정확도 확인)
- 다양한 기기에서 테스트

### 2. 분석 도구 (선택)
- Google Analytics 또는 Vercel Analytics 추가
- 에러 트래킹 (Sentry 등)

### 3. 문서화
- 이용 약관 (선택)
- 개인정보 처리방침 (GPS 데이터 수집)
- 게임 가이드 (이미 있음 - 룰북)

### 단계 1: 소프트 런칭 (추천)

- 이미 충분히 테스트했고

→ **1-2일 후 배포**
