### Pencil 작업 순서

#### 1. 헤더 만들기
- Rectangle (375 x 60px)
- 배경: 흰색 (#FFFFFF)
- 하단 Border: 1px #E5E7EB
- 텍스트 "방 설정" (중앙 정렬, Bold, 18px)
- 왼쪽 아이콘 "<" (뒤로 가기)
- 오른쪽 "완료 ✓" (파란색 #2563EB)

#### 2. 섹션 타이틀
- 텍스트 "📋 기본 설정" (Bold, 16px, 왼쪽 정렬)
- 위쪽 여백: 20px

#### 3. 입력 필드
- Rectangle (335 x 50px)
- 배경: 흰색
- Border Radius: 12px
- Border: 1px #E5E7EB
- 안쪽 여백: 16px
- 왼쪽 레이블 + 오른쪽 입력값

#### 4. 라디오 버튼 그룹
- Circle (20px) - 테두리만
- 선택 시: 안쪽에 작은 원 (10px, 채움)
- 옆에 텍스트

#### 5. 슬라이더
- Line (300px)
- Circle (20px) - 드래그 가능 표시
- 양 끝에 값 표시 (0.5km ~ 5km)

#### 6. 버튼
- Rectangle (335 x 56px)
- 배경: #2563EB (파란색)
- Border Radius: 16px
- 텍스트: "방 만들기" (흰색, Bold, 18px)

## 🎨 Pencil 작업 팁

### 색상 팔레트 저장
1. View > Color Palette
2. 주요 색상 추가:
   - #2563EB (경찰)
   - #DC2626 (도둑)
   - #F9FAFB (배경)
   - #FFFFFF (카드)
   - #E5E7EB (테두리)

### 컴포넌트 재사용
1. 자주 쓰는 요소 (버튼, 카드) 만들기
2. Edit > Copy / Paste
3. 그룹화: Cmd+G

### 레이어 정리
1. 각 화면별로 레이어 그룹
2. "헤더", "본문", "버튼" 등으로 네이밍
3. 순서 정리 (위에서 아래로)

### 정렬 및 간격
1. Pencil의 정렬 도구 사용
2. 여백 일관성 유지:
   - 좌우 여백: 20px
   - 요소 간 간격: 16px
   - 섹션 간 간격: 24px
