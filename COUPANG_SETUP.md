# 🛒 쿠팡 파트너스 설정 가이드

**시기**: 출시 후 언제든지  
**소요 시간**: 10분 + 승인 대기 (1-2일)

---

## 📋 설정 순서

1. 파트너스 가입 (1-2일 승인 대기)
2. 배너 생성
3. 코드 복사
4. 앱에 적용
5. 수익 확인!

---

## 1️⃣ 파트너스 가입

### 가입 페이지
https://partners.coupang.com

### 필요한 정보
- 이름
- 이메일
- 전화번호
- 블로그/웹사이트 URL (경도 앱 URL 입력)
- 은행 계좌 (정산용)

### 승인 대기
- 1-2일 소요
- 이메일로 승인 알림

---

## 2️⃣ 배너 생성

### 로그인 후
1. "링크생성" 메뉴 클릭
2. "배너" 탭 선택

### 배너 설정
- **크기**: 
  - 모바일: 320x50 (추천)
  - PC: 728x90
- **카테고리**: 
  - "전체 상품" 또는
  - "인기 상품"
- **기간**: 자동 갱신

### 코드 생성
"코드 생성" 버튼 클릭 → HTML/JavaScript 코드 표시

---

## 3️⃣ 코드 적용

### 생성된 코드 예시
```html
<script src="https://ads-partners.coupang.com/g.js"></script>
<script>
  new PartnersCoupang.G({
    id: 123456,
    template: "carousel",
    trackingCode: "AF1234567",
    width: "320",
    height: "50"
  });
</script>
```

### j-mac에게 전달
텔레그램으로 전체 코드 전송:
```
쿠팡 배너 코드:

[여기에 복사한 코드 붙여넣기]
```

### 적용 완료
j-mac이 `CoupangBanner.tsx` 파일에 코드 적용 → 재배포

---

## 4️⃣ 수익 확인

### 대시보드
https://partners.coupang.com 로그인

### 확인 항목
- 클릭 수
- 전환율
- 수수료 발생액
- 정산 예정액

### 정산
- 매월 말일 기준
- 익월 10일 지급
- 최소 정산 금액: ₩5,000

---

## 💰 예상 수익

### 계산식
```
일일 사용자 100명
× 클릭률 5%
× 전환율 2%
× 평균 수수료 ₩1,000
= 일 ₩100

월 약 ₩3,000
```

### 사용자 증가 시
```
일일 사용자 1,000명
→ 월 약 ₩30,000
```

## 📝 팁

### 배너 위치 최적화
- 현재: 메인, 대기실, 결과 화면 하단
- 테스트 후 위치 조정 가능
- 게임 방해 최소화

### 클릭률 높이기
- 관련 카테고리 선택 (예: 스포츠, 레저)
- 계절 상품 (여름: 캠핑, 겨울: 보드)
- A/B 테스트 (다양한 배너 시도)

### 주의사항
- 쿠팡 파트너스 약관 준수
- 부적절한 유도 클릭 금지
- 자동 클릭 프로그램 사용 금지
