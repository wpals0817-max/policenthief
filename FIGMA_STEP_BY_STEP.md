## ✅ 준비 단계

### 1. Figma 계정 생성
1. https://www.figma.com 접속
2. **Sign up** 클릭
3. Google 계정으로 로그인 (가장 빠름)
4. 이메일 확인

### 2. 새 디자인 파일 생성
1. 로그인 후 메인 화면
2. **New Design File** 클릭 (좌측 상단 + 버튼)
3. 파일명: **"경도 앱 디자인 v1"**

## 📱 iPhone 프레임 설정

### 1. Frame 도구 선택
- 왼쪽 도구바에서 **Frame (F)** 클릭
- 또는 단축키: **F**

### 2. iPhone 프레임 선택
1. 오른쪽 패널 **Frame** 섹션
2. **Phone** 카테고리 찾기
3. **iPhone 14 Pro** 선택 (393 x 852)
   - 또는 **Custom**: 375 x 812 (iPhone 13)

### 3. 프레임 3개 만들기
1. 첫 프레임 만들기
2. **Alt + 드래그**로 복제
3. 총 3개 나란히 배치
4. 각각 이름 변경:
   - "방 설정 화면"
   - "대기실 화면"
   - "게임 화면"

## 🎨 색상 팔레트 설정

### 1. Local Styles 만들기
1. 오른쪽 패널에서 **Fill** 클릭
2. **+** (새 스타일 추가)
3. 색상 코드 입력:

```
Primary (경찰):    #2563EB
Secondary (도둑):  #DC2626
Background:        #F9FAFB
Card:              #FFFFFF
Border:            #E5E7EB
Text:              #111827
Text Light:        #6B7280
```

### 2. 스타일 저장
각 색상마다:
1. 색상 코드 입력
2. **Create style** 클릭
3. 이름: "Primary", "Secondary" 등

## 🎨 디자인 팁

### 정렬 및 간격
1. 요소 선택
2. 오른쪽 패널에서 **Align** 도구 사용
3. 일관된 간격 유지:
   - 좌우 여백: 20px
   - 요소 간: 12-16px
   - 섹션 간: 24px

### 그룹화
1. 관련 요소 선택 (Shift + 클릭)
2. **Cmd+G** (그룹화)
3. 이름 변경 (더블클릭)

### 컴포넌트화 (재사용)
1. 자주 쓰는 요소 선택
2. **Create Component** (Cmd+Alt+K)
3. 이름: "Button", "Card" 등
4. 다른 곳에서 재사용

### Auto Layout (고급)
1. 그룹 선택
2. **Shift+A** (Auto Layout 적용)
3. 자동 간격 조정
