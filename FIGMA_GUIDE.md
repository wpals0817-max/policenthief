# 🎨 Figma로 경도 앱 디자인하기

**날짜**: 2026-02-15  
**대안**: Pencil 연결 문제로 Figma 사용

---

## 🚀 Figma 빠른 시작

### 1. Figma 접속
```
https://www.figma.com
```

### 2. 무료 계정 생성
- Google 계정으로 로그인 (가장 빠름)
- 또는 이메일로 가입

### 3. 새 파일 생성
1. **New Design File** 클릭
2. 파일명: "경도 앱 디자인 v1"

### 4. iPhone 프레임 추가
1. 왼쪽 도구에서 **Frame (F)** 선택
2. 오른쪽 패널에서 **Phone** 선택
3. **iPhone 14 Pro** (393 x 852) 선택
   - 또는 Custom: 375 x 812

---

## 🎨 기본 사용법 (5분이면 충분)

### 도구 단축키
- **R**: 사각형 (Rectangle)
- **T**: 텍스트 (Text)
- **O**: 원 (Circle)
- **F**: 프레임 (Frame)
- **V**: 선택 (Select)
- **Cmd+D**: 복제

### 색상 팔레트 설정
1. 오른쪽 **Fill** 클릭
2. 색상 코드 입력:
   - `#2563EB` (경찰 파란색)
   - `#DC2626` (도둑 빨간색)
   - `#F9FAFB` (배경)

### 빠른 작업 순서
1. **프레임 만들기**: iPhone 프레임
2. **배경 추가**: 사각형 (R) + 색상
3. **텍스트 추가**: 텍스트 (T)
4. **버튼 만들기**: 사각형 + 텍스트 + Border Radius
5. **복제**: Cmd+D로 반복

## 📱 경도 앱 3가지 화면 만들기

### 화면 1: 방 설정
1. iPhone 프레임 3개 나란히 배치
2. 첫 번째 프레임:
   - 헤더 (사각형 60px 높이)
   - 입력 필드 (사각형 + 텍스트)
   - 버튼 (사각형 56px 높이)

### 화면 2: 대기실
1. 두 번째 프레임:
   - 방 코드 박스
   - 2개 컬럼 (경찰/도둑)
   - 플레이어 카드

### 화면 3: 게임 화면
1. 세 번째 프레임:
   - 상태 바
   - 지도 (간단한 사각형)
   - 정보 박스

## 💾 완성 후 내보내기

### PNG로 저장
1. 프레임 선택
2. 오른쪽 하단 **Export** 클릭
3. **PNG** 선택
4. **Export** 버튼

### 공유 링크
1. 우측 상단 **Share** 버튼
2. **Copy link** 클릭
3. 링크 공유 (누구나 볼 수 있음)
